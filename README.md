# Cognitia AI

Cognitia AI is an intelligent academic learning platform designed for competitive exam aspirants (JEE & NEET).  

## 🚀 Features

- AI-powered interactive blackboard
- Real-time lesson streaming (WebSocket)
- Adaptive quizzes
- Exam-based topic filtering
- Progress tracking & gamification
- Multi-language support

## 🛠 Tech Stack

Frontend:
- Next.js
- TypeScript
- Tailwind CSS
- ShadCN UI

Backend:
- FastAPI
- WebSockets
- SQLAlchemy
